# OmenXHub-Main 项目完善总结

## 项目概述
OmenXHub-Main 是基于 OmenSuperHub-Master 的 WPF 重构版本，实现了类似的核心功能，同时提供了更现代化的用户界面和更好的架构设计。

## 架构设计

### 项目结构
```
OmenXHub-main/
├── App/                    # 应用层
│   ├── Strings.cs          # 多语言支持
│   ├── GpuAppManager.cs    # GPU 程序管理
│   ├── Logger.cs           # 日志系统
│   └── OmenLighting.cs     # 灯光控制
├── Services/               # 服务层
│   ├── ConfigService.cs    # 配置管理
│   ├── FanService.cs       # 风扇服务
│   ├── HardwareService.cs  # 硬件服务
│   ├── ThemeService.cs     # 主题服务
│   └── TrayService.cs      # 托盘服务
├── Views/                  # 视图层（WPF）
│   ├── MainWindow.xaml/.cs # 主界面
│   ├── FloatingWindow.xaml/.cs # 浮窗
│   └── HelpWindow.xaml/.cs # 帮助窗口
├── OmenHardware.cs         # 硬件控制核心
└── Resources/              # 资源文件
```

- 注册表持久化存储

- 多语言设置
- 浮窗配置
- 自动启动设置

#### 2. FanService（风扇控制）
- 温度-转速映射表

- 自定义曲线编辑

- LibreHardwareMonitor 集成
- GPU 连接状态检测
- 温度/功耗监控
- 自动监控开关

#### 4. OmenHardware（硬件控制）
- WMI 通信
- 风扇控制
- 性能模式设置
- CPU/GPU 功率控制
- 灯光控制
- 图形模式切换

## 使用说明

### 安装运行
1. 依赖 .NET Framework 4.8
2. 首次运行自动初始化配置
3. 支持系统启动自启

### 主要功能使用
1. **预设切换**: 在"总览"页面选择预设方案
2. **风扇控制**: 在"风扇"页面自定义转速曲线
3. **性能控制**: 在"性能"页面调整各项设置
4. **灯光控制**: 在"灯光"页面设置键盘/灯条效果
5. **系统信息**: 在"本机信息"页面查看硬件状态

### 配置文件
- 注册表存储：`HKEY_CURRENT_USER\Software\OmenXHub`
- 预设存储：`HKEY_CURRENT_USER\Software\OmenXHub\Presets\[PresetName]`
- 风扇曲线：`AppDomain.CurrentDomain.BaseDirectory\[filename].txt`

## 性能优化

### 已实现的优化
1. **WMI 调用优化**: 减少不必要的调用
2. **温度平滑过渡**: 降低 CPU/GPU 监控频率
3. **自动监控开关**: GPU 空闲时自动停止监控
4. **内存管理**: 定期清理和释放资源
5. **预设管理**: 只保存必要的配置项

### 未来优化方向
1. 数据库持久化（可选）
2. 云端配置同步
3. 历史记录追踪
4. 性能分析工具集成

## 安全性考虑

1. **权限管理**: 只使用用户级权限访问注册表
2. **错误处理**: 全面的异常捕获和处理
3. **驱动嵌入**: DB 驱动作为资源嵌入，减少外部依赖
4. **日志记录**: 完整的操作日志
5. **版本检测**: 驱动版本限制防止兼容性问题
